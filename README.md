# Password Strength Checker

A modern, responsive, and beginner-friendly Password Strength Checker web application. Built entirely with Vanilla HTML, CSS, and JavaScript, it is ready to be hosted instantly on GitHub Pages.

Live Demo: https://vamsirajoleti.github.io/Password-checker-/

## Features

- **Real-Time Validation:** Checks password requirements instantly as you type.
- **Dynamic Visual Feedback:** Animated progress bar and strength labels (Red/Orange/Yellow/Green).
- **Regex Rule Checking:** Ensures minimum length, uppercase, lowercase, numbers, and special characters.
- **Dark/Light Mode:** Includes a theme toggler that defaults to system preference.
- **Password Generator:** A single click creates a highly secure, 16-character randomized password.
- **Copy to Clipboard:** Easily copy the typed or generated password with visual confirmation.
- **Fully Responsive:** Looks great on desktop, tablets, and mobile devices.

## How to Run

Since this project relies on core web technologies without any heavy frameworks:

1. Clone or download this repository.
2. Unzip the folder.
3. Open `index.html` in any modern web browser.
4. (Optional) To host online, just push these files to a public GitHub repository and enable **GitHub Pages** from your repository settings.

## File Structure

- `index.html` - The structure of the page and embedded SVGs for icons.
- `style.css` - All styling, theme variables (Dark/Light mode), and layout.
- `script.js` - Logic handling for Regex validation, UI state updates, theme toggling, and button actions.
